项目构成

项目是由模型和可视化两部分构成

模型部分

<AntiScam>资源集

  <data>数据

     <class.txt>分类集 <train.txt>训练集 <test.txt>测试集 <dev.txt>验证集 <vocab.pkl>word_to_id表

     <embedding_SougouNews.npz>预训练embedding

  <log>训练日志

  <saved_dict>训练过的模型文件

<model>模型代码文件

<train_eval.py>训练、测试、评估等函数文件

<utils.py>Dateset、Dataloader、词表等函数文件

<utils_fasttext.py>fasttext专用、作用同utils.py

<run.py>模型训练的启动文件

可视化部分

<app.py>flask核心文件，项目后端

<templates>前端html文件

<static>前端css、js、image及各种插件

项目运行

Tip:

可以先自行梳理代码结构，争取看懂代码

模型训练

>>> python run.py [-h] --model MODEL [--embedding EMBEDDING] --cuda CUDA [--word WORD] --note NOTE
--model 'choose a model: TextCNN, TextRNN, FastText, TextRCNN, TextRNN_Att, DPCNN, Transformer'
--embedding 'random or pre_trained'
--cuda 'choose a cuda: 0, 1'
--word 'True for word, False for char'
--note 'write something to remark'

可视化运行

>>>python app.py
